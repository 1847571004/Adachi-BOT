## Usage
### Install
```
git clone https://github.com/SilveryStar/Adachi-BOT.git
cd Adachi-BOT
npm install
```
### Config
```
> setting.yml
account:
  qq: your qq number
  password: your password or password md5
master: your master qq number
  
> cookies.yml
cookies:
  - your mihoyoBBS cookies
```

### Run
```
npm start
```
